# Integração Trello x Asana - SysMiddle

Projeto desenvolvido em Python para integrar informações entre Trello e Asana, com foco em automação de fluxos, leitura de dados e envio de informações entre plataformas de gerenciamento de tarefas.

## Objetivo

Este script foi criado para apoiar processos de integração entre ferramentas de gestão, permitindo centralizar informações e facilitar a sincronização de dados entre boards do Trello e estruturas do Asana.

## Funcionalidades

- Conexão com a API do Trello
- Conexão com a API do Asana
- Consulta de informações em boards/listas/cartões
- Envio de dados para o Asana
- Organização de variáveis sensíveis por ambiente
- Estrutura simples para evolução e manutenção

## Tecnologias utilizadas

- Python
- Requests
- Python Dotenv
- APIs REST
- Trello API
- Asana API
